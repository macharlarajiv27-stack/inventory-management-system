# Inventory Management System

## Overview
A Java Backend Inventory Management System built using Hibernate ORM and MySQL.

## Technologies Used
- Java
- Hibernate ORM
- JPA
- MySQL
- Maven

## Features
- Product Management
- Supplier Management
- Inventory Tracking
- One-to-One Mapping
- One-to-Many Mapping
- Many-to-One Mapping
- Lazy Fetching
- Eager Fetching
- Hibernate Caching

## Project Structure

src/main/java/com/rajiv
- Product.java
- Supplier.java
- InventoryDetail.java
- HibernateUtil.java
- Main.java

## Concepts Implemented
- Entity Mapping
- Relationship Mapping
- Fetch Types
- Hibernate Session Management
- Caching
- CRUD Operations
